# Kali-Hunter 🏹
أداة متخصصة في فحص الشبكات واختبار الاختراق، مصممة للعمل على بيئة Termux.

## 🛠 التثبيت (Installation)
لانتحال أو تشغيل الأداة، استخدم الأوامر التالية:

```bash
pkg update && pkg upgrade
pkg install git python -y
git clone https://github.com/kaly66614-sketch/Kali-Hunter.git
cd Kali-Hunter
python kali_hunter.py
```
